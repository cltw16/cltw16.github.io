Wie erstellt man einen GNUPG Key?

Zuerst ist es wichtig das man einen Browser wie Thunderbird benutzt. Outlook ist nur in der 32bit Version geeignet. Diese Website hat ein Plugin für Thunderbird bereitgestellt. Installieren sie dies und folgen sie der Anleitung. Achten sie darauf das ihre Passphrase besonders stark ist. benutzen sie einen Keyverwalter wie Kleopatra dann können sie nun den GPG Key sehen.
